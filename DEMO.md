# BotUz v2 — Демонстрация всех функций

```bash
cd botuz
npm install
npm run dev
# Открыть http://localhost:3000
```

## Демо-сценарии

### 1. Лендинг (http://localhost:3000)

**Что увидите:**
- Hero-секция с анимированным заголовком и CTA "Создать бота бесплатно"
- Переключатель языков (RU / UZ / EN)
- Секция "Как это работает" — 3 шага с иконками
- Секция "Для кого" — 9 ниш бизнеса с эмодзи
- Живой чат-превью — анимация диалога бота с клиентом
- Footer с ссылками

### 2. Визард — Создание бота (http://localhost:3000/create)

**10 шагов Typeform-стиля:**

| Шаг | URL | Что проверить |
|-----|-----|--------------|
| 1 | /create | Выбор языка интерфейса (3 карточки, auto-advance) |
| 2 | | Выбор языка бота (1 или 2 языка, тарификация) |
| 3 | | Сфера бизнеса (3x3 grid с эмодзи) |
| 4 | | Возможности бота (toggle-карточки, пресеты по нише) |
| 5 | | Информация о бизнесе (форма: название, описание, часы) |
| 6 | | Товары/услуги (4 способа добавления) |
| 7 | | База знаний / FAQ (предзаполненные по нише) |
| 8 | | Подключение платформ (Telegram токен) |
| 9 | | Тон бота + приветствие + live preview |
| 10 | | Финальный обзор + кнопка "Запустить бота!" |

**Проверить:**
- [ ] Прогресс-бар сверху
- [ ] Кнопка "Назад" на каждом шаге
- [ ] Slide-анимация переходов
- [ ] Данные сохраняются в localStorage (обновите страницу — данные на месте)
- [ ] Конфетти при запуске бота (шаг 10)

**Демо-путь:**
1. Русский → 2. Русский+Узбекский → 3. Магазин одежды → 4. Каталог+Заказы+AI → 5. "Bella Moda" → 6. Добавить 2-3 товара → 7. Проверить FAQ → 8. Пропустить Telegram → 9. Дружеский тон → 10. Запуск

### 4. API эндпоинты (http://localhost:3000/api/trpc)

| Эндпоинт | Описание |
|-----------|---------|
| bot.create | Создание бота из визарда |
| bot.getById | Получение бота по ID |
| product.list | Список товаров бота |
| product.create | Добавление товара |
| faq.list | Список FAQ |
| faq.generateForNiche | Генерация FAQ по нише |
| order.list | Список заказов |
| order.updateStatus | Обновление статуса |
| conversation.list | Список диалогов |
| wizard.saveStep | Сохранение шага визарда |
| wizard.complete | Финализация и создание бота |

---

### 5. AI-движок (серверная часть)

**Файлы:**
- `src/server/services/ai-engine.ts` — обработка сообщений через OpenAI
- `src/server/services/prompt-builder.ts` — динамический system prompt
- `src/server/services/faq-templates.ts` — шаблоны FAQ по 4 нишам
- `src/server/services/telegram-bot.ts` — grammY интеграция

**Функции AI:**
- Автоопределение языка клиента (ru/uz)
- 3 режима тона (деловой/дружеский/весёлый)
- Включение каталога товаров в промт
- Включение FAQ в промт
- Определение intent (order, faq, handoff)
- Автоматическая передача менеджеру при низком confidence

---

### 6. i18n — Мультиязычность

**3 языка:** Русский, Узбекский (латиница), English
**Файлы:** `src/i18n/locales/{ru,uz,en}.json`

Переводы покрывают:
- Лендинг (все тексты)
- Визард (все 10 шагов)
- Дашборд (меню, метрики, действия)
- Типы бизнеса, возможности бота, статусы заказов

```

├── prisma/
│   ├── schema.prisma          # 11 моделей, 4 enum'а
│   └── seed.ts                # Демо-данные (Bella Moda + Opi Plov)
├── src/
│   ├── app/
│   │   ├── page.tsx            # Лендинг

│   │   └── api/                # tRPC + Telegram webhook
│   ├── components/
│   │   ├── landing/            # 5 компонентов лендинга
│   │   ├── wizard/             # 12 компонентов визарда
│   │   └── dashboard/          # 9 компонентов дашборда
│   ├── server/
│   │   ├── trpc.ts             # tRPC инициализация
│   │   ├── routers/            # 7 API роутеров
│   │   └── services/           # AI engine, Telegram, FAQ templates
│   ├── i18n/locales/           # 3 языка
│   ├── lib/                    # Prisma client, tRPC client, demo data

└── __tests__/                  # Unit тесты
```

**Всего:** 55+ исходных файлов, 12 маршрутов, 11 моделей БД
